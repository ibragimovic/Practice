1. Модифкатор доступа - это определение доступа из одного класса к свойствам, методам и конструкторам другого класса,
где один класс может находиться в одном пакете (директория, где хранится класс), а второй класс может находиться в
в другом пакете.
2. Модификаторы доступа бывают public, private, default, protected.
3. Package (пакет) - это директория (папка) в которой хранятся классы, они нужны чтобы не было конфликтов
при наименовании классов, а также для удобства при поиске на компьюторе. Если класс к которому обращаются находится в
другом пакете нужно написать import + директорию.
4. Модификатор доступа default - обеспечивает доступ к свойствам, методам и конструкторам только внутри текущего пакета,
а модификтор доступа public обеспечивает доступ из любой точки программы, из любого пакета и из любого класса.
5. Для того чтобы пользователь мог получить определенные параметры, а присваивать значения не мог.
